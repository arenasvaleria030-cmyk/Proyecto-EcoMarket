#  ekomarket

**ekomarket** es una plataforma de comercio electrónico diseñada para conectar a consumidores con productores locales de productos ecológicos y sostenibles.

---

## Características

*  Catálogo de productos ecológicos y Km 0.
*  Panel para que los productores gestionen sus productos.
*  Pasarela de pagos segura y fácil de usar.
